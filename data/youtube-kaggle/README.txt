Data
This dataset includes several months (and counting) of data on daily trending YouTube videos.
Data includes
video title,
channel title,
publish time,
tags,
views,
likes and dislikes,
description and
comment count

The data also includes a category_id field, which varies between regions. To retrieve the categories for a specific video, find it in the associated JSON.
https://www.kaggle.com/datasnaek/youtube-new
